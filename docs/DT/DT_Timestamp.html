static parse [line 64]

static DT_Timestamp parse(string $text, [DT_Format $format = NULL])

指定されたテキストを解析して DT_Timestamp オブジェクトに変換します.

$format が指定されていない場合は DT_W3cDatetimeFormat::getInstance() を使って解析を行います. ("YYYY-MM-DD hh:mm:ss" 形式の文字列を受理します. 日付と時刻のセパレータは, 数字以外の ASCII 1 文字であれば何でも構いません.)
