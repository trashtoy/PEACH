Arrays クラスの概要

Util_Arrays は, 配列のソートや最大値・最小値の取得など, 配列操作に関する汎用処理を集めたユーティリティクラスです.

オブジェクトの配列をソートする

オブジェクトを含んだ配列をソートするための方法として, 一般的に usort が利用されますが, このモジュールでは Java のコレクション・フレームワークのようなオブジェクト指向的なソート API を使うことができます. オブジェクトの配列をソートする際に利用する Util_Comparator および Util_Comparable は, それぞれ java.util.Comparator, java.util.Comparable と同じような使い方をします.

Util_Comparable の使用例

Util_Comparable を使ったソートの例を以下に示します. ソート対象のクラスで implements Util_Comparable を宣言して, compareTo() メソッド内にソート条件を定義します.

  1. class TestObject implements Util_Comparable {
  2.     private $id;
  3.     private $value;
  4.  
  5.     public function __construct($id, $value) {
  6.         $this->id    = $id;
  7.         $this->value = $value;
  8.     }
  9.  
  10.     public function getId() {
  11.         return $this->id;
  12.     }
  13.  
  14.     public function getValue() {
  15.         return $this->value;
  16.     }
  17.  
  18.     /**
  19.      * 引数の TestObject の value を比較します.
  20.      * value が等しい場合は id を比較します.
  21.      */
  22.     public function compareTo($subject) {
  23.         if ($subject instanceof TestObject) {
  24.             $comp = $this->value - $subject->value;
  25.             if (0 < $comp) return 1;
  26.             if ($comp < 0) return -1;
  27.  
  28.             $comp2 = $this->id - $subject->id;
  29.             if (0 < $comp2) return 1;
  30.             if ($comp2 < 0) return -1;
  31.  
  32.             return 0;
  33.         } else {
  34.             throw new Exception();
  35.         }}
  36. }
  37.  
  38. $arr = array(
  39.     new TestObject(1, 250),
  40.     new TestObject(5, 100),
  41.     new TestObject(4, 400),
  42.     new TestObject(3, 320),
  43.     new TestObject(2, 400)
  44. );
  45.  
  46. $arr = Util_Arrays::sort($arr); // => ID が 5(100), 1(250), 3(320), 2(400), 4(400) の順にソートされる

Util_Comparator の使用例

Util_Comparable を使った例では既存のクラスに手を加えることでソートを実装しましたが, 他者が公開したクラスライブラリのように自由に改変することが難しいクラスをソートする場合や, ソート条件を状況に応じて柔軟に切り替えたい場合には Util_Comparator を使ってください. Util_Comparator を使ったソートの例を以下に示します. この例では一つの配列に対して, 4種類の Comparator を使ってソートを行います. それぞれ別の順序でソートされることに着目してください.

  1. class TestObject {
  2.     private $id;
  3.     private $value;
  4.  
  5.     public function __construct($id, $value) {
  6.         $this->id    = $id;
  7.         $this->value = $value;
  8.     }
  9.  
  10.     public function getId() {
  11.         return $this->id;
  12.     }
  13.  
  14.     public function getValue() {
  15.         return $this->value;
  16.     }
  17. }
  18.  
  19. /**
  20.  * 1.value 昇順, 2.ID 昇順の順番でソートします
  21.  */
  22. class TestObjectComparator1 implements Util_Comparator {
  23.     public function compare($var1, $var2) {
  24.         if (($var1 instanceof TestObject) && ($var2 instanceof TestObject)) {
  25.             $comp = $var1->getValue() - $var2->getValue();
  26.             if (0 < $comp) return 1;
  27.             if ($comp < 0) return -1;
  28.  
  29.             $comp2 = $var1->getId() - $var2->getId();
  30.             if (0 < $comp2) return 1;
  31.             if ($comp2 < 0) return -1;
  32.  
  33.             return 0;
  34.         } else {
  35.             throw new Exception();
  36.         }}
  37. }
  38.  
  39. /**
  40.  * 1.ID 昇順, 2.value 昇順の順番でソートします
  41.  */
  42. class TestObjectComparator2 implements Util_Comparator {
  43.     public function compare($var1, $var2) {
  44.         if (($var1 instanceof TestObject) && ($var2 instanceof TestObject)) {
  45.             $comp = $var1->getId() - $var2->getId();
  46.             if (0 < $comp) return 1;
  47.             if ($comp < 0) return -1;
  48.  
  49.             $comp2 = $var1->getValue() - $var2->getValue();
  50.             if (0 < $comp2) return 1;
  51.             if ($comp2 < 0) return -1;
  52.  
  53.             return 0;
  54.         } else {
  55.             throw new Exception();
  56.         }}
  57. }
  58.  
  59. $c1 = new TestObjectComparator1();
  60. $c2 = new TestObjectComparator2();
  61. $r1 = new Util_ReverseComparator($c1);
  62. $r2 = new Util_ReverseComparator($c2);
  63.  
  64. $arr = array(
  65.     new TestObject(5, 100),
  66.     new TestObject(3, 300),
  67.     new TestObject(2, 100),
  68.     new TestObject(4, 200),
  69.     new TestObject(1, 300)
  70. );
  71.  
  72. $result1 = Util_Arrays::sort($arr, $c1); // ID が 2(100), 5(100), 4(200), 1(300), 3(300) の順にソート
  73. $result2 = Util_Arrays::sort($arr, $r1); // ID が 3(300), 1(300), 4(200), 5(100), 2(100) の順にソート
  74. $result3 = Util_Arrays::sort($arr, $c2); // ID が 1, 2, 3, 4, 5 の順にソート
  75. $result4 = Util_Arrays::sort($arr, $r2); // ID が 5, 4, 3, 2, 1 の順にソート

max, min の活用方法

配列の中から最大 (あるいは最小) の項目を抜き出すというケースはよくありますが, 普通は

  1. 配列をソートする
  2. 配列の先頭 (あるいは末尾) の項目を取り出す

という手順で実現されることが多いです.

そのようなケースでは Util_Arrays::max() あるいは Util_Arrays::min() を使うことで簡単に最大・最小値を取り出すことが出来ます.

(数値の配列から最大・最小値を取り出す場合はビルトイン関数の min(), max() を使ってください. ただし, パフォーマンスは劣りますが Util_Arrays::max(), Util_Arrays::min() でも同様の結果を得られます.)

  1. class TestObject implements Util_Comparable {
  2.     // (中略)
  3. }
  4.  
  5. $arr = array(
  6.     new TestObject(1, 250),
  7.     new TestObject(5, 100),
  8.     new TestObject(4, 400),
  9.     new TestObject(3, 320),
  10.     new TestObject(2, 400)
  11. );
  12.  
  13. $max = Util_Arrays::max($arr); // => ID = 4, value = 400 のオブジェクトを取得する
