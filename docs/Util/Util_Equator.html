hashCode [line 33]

int hashCode(mixed $var)

指定された値のハッシュ値を計算します.

もしもこのオブジェクトの equate($var1, $var2) が TRUE を返す場合, hashCode($var1) と hashCode($var2) が必ず同じ値を返すようにしなければなりません.

この制約条件は Java の java.lang.Object#equals(), java.lang.Object#hashCode() で定義されている規約に相当します.
