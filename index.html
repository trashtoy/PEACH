PEACH

PEACH とは, PHP プログラミングをラクにして, 保守性の高いコードを書くための汎用 PHP クラスライブラリです. 以下の分野で活躍します.

  • Util : 配列操作・その他汎用的な用途
  • DT : 日付・時刻の操作
  • Markup : XML, HTML の出力支援

利用するには, 以下のように autoload.php を require すれば OK です.

  1. require_once("/path/to/PEACH/autoload.php");
